WOODLED Shortener — Tester v2

Три варианта POST. Ищем тот, который проходит CORS из браузера на Apps Script.

POST варианты

A. text/plain (классический, как был)

Content-Type: text/plain;charset=utf-8 — должен быть simple request без preflight.

B. application/x-www-form-urlencoded

Отправляет form-data с полем "data" в котором JSON-строка. В Apps Script читается из e.parameter.data.

C. FormData (multipart)

Отправляет multipart/form-data. В Apps Script читается из e.parameter.data.

D. fetch без явного Content-Type

Браузер сам определяет Content-Type для строки (обычно text/plain;charset=UTF-8).
